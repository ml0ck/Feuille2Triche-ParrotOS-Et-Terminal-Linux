Cheat Sheet : Utilisation de SSH et Sécurisation Réseau

Guide Complet pour Utiliser SSH

Étape 1 : Installer SSH

Pour installer le serveur SSH sur votre machine Linux, utilisez la commande suivante :

sudo apt update && sudo apt install openssh-server

Après l'installation, vérifiez que le service SSH fonctionne :

sudo systemctl status ssh

Étape 2 : Se connecter à une Machine Distante

Pour se connecter à une machine distante via SSH, utilisez cette commande :

ssh nom_utilisateur@adresse_IP

Exemple :

ssh john@192.168.1.100

Entrez le mot de passe pour accéder à la machine distante.

Étape 3 : Générer des Clés SSH pour une Connexion sans Mot de Passe

Pour une connexion sans mot de passe, générez des clés SSH avec :

ssh-keygen -t rsa -b 4096

Puis, copiez votre clé publique vers la machine distante avec :

ssh-copy-id nom_utilisateur@adresse_IP

Vous pourrez vous connecter sans entrer de mot de passe après cette étape.

Étape 4 : Renforcer la Sécurité de votre Serveur SSH

Pour sécuriser votre serveur SSH, éditez le fichier /etc/ssh/sshd_config :

  • Désactiver l'accès root : PermitRootLogin no
  • Changer le port par défaut : Port 2222
  • Désactiver l'authentification par mot de passe : PasswordAuthentication no

Redémarrez SSH après ces modifications :

sudo systemctl restart ssh

Étape 5 : Transférer des Fichiers avec SCP

Utilisez SCP pour transférer des fichiers via SSH :

scp fichier_local nom_utilisateur@adresse_IP:chemin_destination

Exemple :

scp fichier.txt john@192.168.1.100:/home/john/
